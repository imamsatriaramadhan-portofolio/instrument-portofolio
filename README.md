# Imam Satria Ramadhan Instrumentation Portfolio

GitHub Pages-ready static portfolio.

## Update content
- Edit `data.js` to add new case studies.
- Put photos/videos inside `assets/`.
- Add media inside a case using:

```js
media:[
  {type:'image', src:'assets/example.jpg', caption:'Field documentation'},
  {type:'video', src:'assets/example.mp4', caption:'Calibration video'}
]
```

## Publish
Upload all files/folders to GitHub repository root and enable GitHub Pages from branch `main`, folder `/root`.
